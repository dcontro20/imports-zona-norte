# IMPORTS ZONA NORTE — Contexto Completo para Claude Code

## 1. RESUMEN DEL NEGOCIO

**Imports Zona Norte** es un negocio de importación y venta minorista de vapes (cigarrillos electrónicos) en Argentina. Los socios son **Diego** y **Gustavo**.

- Importan vapes desde **Paraguay**, pagando en **USDT** (cripto)
- Venden minorista en Argentina por **WhatsApp, Instagram, Delivery, Presencial**
- Cobran en: **Mercado Pago, Lemon (pesos y USDT), USD Cash, USDT, Pesos Cash**
- Volumen: ~80-100 unidades por mes entre los dos
- Ganancia promedio: $7-10 USD por vape → ~$680 USD bruto/mes → $340 c/u

### Fórmula de costo real por unidad
```
costoReal = (precioUSDT_proveedor × 1.01 × 1.05) + 0.40 USD envío
```
- 1.01 = comisión proveedor (~1%)
- 1.05 = comisión pasero (~5% del valor ARS)
- 0.40 USD = envío Vía Cargo prorrateado por unidad

### Conversión de precios
- Precios base están en USD
- Precio ARS = Precio USD × cotización dólar blue
- El blue se obtiene automáticamente de `https://dolarapi.com/v1/dolares/blue` cada 10 minutos
- Campo `venta` del JSON de respuesta

## 2. STACK TECNOLÓGICO

| Componente | Tecnología |
|---|---|
| Frontend | React 18 + Vite 5 |
| Backend/DB | Firebase Firestore (real-time sync) |
| Hosting | Vercel (auto-deploy desde GitHub main) |
| Repo | `https://github.com/dcontro20/imports-zona-norte` |
| URL producción | `https://imports-zona-norte.vercel.app` |

### package.json
```json
{
  "name": "imports-zona-norte",
  "private": true,
  "version": "1.0.0",
  "type": "module",
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "preview": "vite preview"
  },
  "dependencies": {
    "react": "^18.2.0",
    "react-dom": "^18.2.0",
    "firebase": "^10.8.0"
  },
  "devDependencies": {
    "@vitejs/plugin-react": "^4.2.1",
    "vite": "^5.1.0"
  }
}
```

### vite.config.js
```javascript
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [react()],
  build: {
    rollupOptions: {
      output: {
        manualChunks: {
          firebase: ['firebase/app', 'firebase/firestore'],
          react: ['react', 'react-dom'],
        }
      }
    }
  }
})
```

### Autenticación
NO usa Firebase Auth. Login simple con contraseñas hardcodeadas en App.jsx:
- **Diego**: password `Poncharelo20!`, color `#6366f1`, icon `💜`
- **Gustavo**: password `Gus2026!`, color `#10b981`, icon `😙`
- Se guarda en `sessionStorage` bajo key `vapestock_user`
